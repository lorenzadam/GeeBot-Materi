Misi 3

Membuat LED dapat Menyala dan Mati dengan Push Button.

Pengenalan

Tujuan

Membuat program agar modul LED yang mati dapat menyala ketika push button ditekan, dan LED yang menyala dapat mati ketika push button ditekan.

Alat yang dibutuhkan

1. Sebuah Arduino

2. Sebuah Shield

3. Sebuah Modul LED

4. Sebuah Modul Push Button

5. 2 buah Kabel

Langkah pertama (Kesepakatan):

Kita harus sepakati terlebih dahulu pin yang akan kita pakai pada arduino untuk misi kali ini adalah pin 2 untuk push button dan pin 13 untuk LED

Langkah kedua (Pemasangan Komponen):

1. Sambungkan Shield dengan Arduino

2. Sambungkan Kabel dengan modul LED

3. Sambungkan Kabel dengan modul Push Button

3. Sambungkan kabel yang telah terhubung dengan modul LED ke PIN 13 dan kabel yang terhubung dengan modul Push button ke PIN 2 pada Shield yang telah terpasang di Arduino

Langkah 4 (Pemrograman)

Masukan kode program dibawah


const int buttonPin = 2;   
const int ledPin =  13; 
int buttonState = 0;   
int ledstate = 0;

void setup() {
  pinMode(ledPin, OUTPUT);
  pinMode(buttonPin, INPUT);
}

void loop() {
  buttonState = digitalRead(buttonPin);
  ledstate = digitalRead(ledPin);
  if ((buttonState == HIGH) && (ledstate == LOW)){
    digitalWrite(ledPin, HIGH);
  } 

  if ((buttonState == HIGH) && (ledstate == HIGH)){
    digitalWrite(ledPin, LOW);
  } 
}

Langkah 5 (Upload):

Agar coding dapat dijalankan oleh arduino, maka kita harus menguploadnya dengan cara :

1. Sambungkan arduino dengan komputer

2. Tekan tombol upload pada program


Program sudah terpasang pada arduino.

Keterangan:

ledstate dideklarasikan karena kita ingin mengetahui status modul led(apakah sedang menyala atau mati).

Keterangan:

"&&" adalah operator yang menyatakan DAN, dimana jika terletak simbol tersebut, maka kedua syarat harus terpenuhi.
