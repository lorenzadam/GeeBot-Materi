Apa Itu Pemrograman?

Pengenalan

Untuk membuat robot, kita harus mampu melakukan pemrograman

Pemrograman adalah proses yang dilakukan untuk memberikan perintah kepada robot dan komputer untuk melakukan sesuatu sesuai yang kita inginkan.

Bagaimana Mereka Dapat Mengerti Perintah Kita?

Perintah yang diberikan kepada Robot harus ditulis menggunakan kode program. Dimana kode program ini adalah bahasa yang dapat komputer mengerti.

Kita dapat membuat banyak hal dapat bekerja sesuai keinginan kita dengan cara mengendalikan komputer yang ada di dalam nya

Smartphone, pintu otomatis, lift ... dan ROBOT!!!

Bagaimana Cara Robot Bekerja?

Kita dapat memberikan perintah pada robot menggunakan kode program.

Bagian paling penting dari membuat robot adalah memberikan perintah akurat. Kenapa? Karena jika perintah tidak akurat maka robot tidak dapat mengerti apa yang harus dilakukan.

Apa maksud dari memberikan perintah yang akurat?

Apakah memberikan perintah pada robot berbeda dengan cara memberikan perintah kepada seseorang?

Apa itu kode program?

Kode program seperti puzzle yang rumit...


#define trigPin 10
#define echoPin 13
 
float duration, distance;
 
void setup() {
  Serial.begin (9600);
  pinMode(trigPin, OUTPUT);
  pinMode(echoPin, INPUT);
}
 
void loop() {  
  digitalWrite(trigPin, LOW);
  delayMicroseconds(2);
  digitalWrite(trigPin, HIGH);
  delayMicroseconds(10);
  digitalWrite(trigPin, LOW);
 
  duration = pulseIn(echoPin, HIGH);
  distance = (duration / 2) * 0.0343;

  Serial.print("Distance = ");
  if (distance >= 400 || distance <= 2) {
     Serial.println("Out of range");
  }
  else {
    Serial.print(distance);
    Serial.println(" cm");
    delay(500);
  }
  delay(500);
}

Lalu bagaimana kita membuat robot?

Untuk dapat membuat robot kamu terlebih dahulu harus paham mengenai algoritma

Algoritma adalah susunan langkah-langkah yang dibuat untuk menyelesaikan suatu perintah

Mari kita lihat contoh kasus kak Rio membutuhkan remote TV tadi

Mari Pecahkan Menjadi Beberapa Langkah

  1. Maju 5 langkah ke depan
  2. Pegang remote dengan tangan kanan
  3. Angkat tangan kanan yang memegang remote
  4. Berbalik
  5. Berjalan menuju kak Rio
  6. Berikan remote kepada kak Rio

Kita sebagai manusia tidak perlu perintah detail seperti itu karena kita mampu membuat keputusan sendiri, namun bagi robot mereka perlu perintah akurat seperti hal tersebut.

Susunan Perintah Tersebut Kita Sebut Dengan Algoritma

Apa langkah selanjutnya?

Setelah kita membuat memikirkan robot tersebut akan dapat melakukan apa, maka langkah selanjutnya adalah mulai dengan membuat kode program.

Mari Mulai

Untuk memulai membangun robot, pastikan 4 hal berikut kamu lakukan:

  1. Kamu menginginkan robot melakukan apa?
  2. Pikirkan setiap langkah kecil yang akan robot tersebut ambil.
  3. Berikan perintah pada robot untuk setiap langkah tersebut.
  4. Lihat apakah robot bekerja sesuai keinginan kamu. Perbaiki jika ada masalah.
